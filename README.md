# 프로젝트 설명

- perfect-pick은 사용자가 원하는 전자제품을 쉽고 빠르게 구매할 수 잇는 전자제품 쇼핑몰 사이트 입니다

- 스마트폰, 태블릿, 노트북, 웨어러블 디바이스를 제공합니다.

- 사용자의 취향 및 요구를 반영한 브랜드 별 셋업 컬렉션을 제공합니다.
